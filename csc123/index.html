For our code, we had each make's city mpg values added up altogether and then took the average for it. The same process was done for the highway mpg values and average. In order to do this, we had to create an array with all the makes listed in the dataset that our code filtered and mapped through.

City and Highway MPG Relation to Fuel Type

Figure Insights

Based on a look at this colored scatter plot, most of the vehicles in this dataset are all in one central area between the 0-50 mpgs. There however appears to be an outlier that runs on compressed natural gas and strays away from the main group of scattered points.

Most of these vehicles run on gasoline. Vehicles running on E85 looks to have lower city and highway mpg's while vehicles running on diesel are higher in both aspects.
